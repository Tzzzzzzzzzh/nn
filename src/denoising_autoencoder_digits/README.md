# 手写数字图像去噪自编码器

本项目实现了一个面向神经网络课程的图像去噪自编码器演示模块。项目不依赖外部数据集，而是使用 8x8 数字模板生成合成手写数字样本，再加入高斯噪声构造“带噪图像 -> 干净图像”的监督学习任务。相比简单均值滤波，自编码器能够学习数字图像的低维结构表示，从而在保留笔画形状的同时去除随机噪声。

## 功能内容

- 构建 0 到 9 的 8x8 数字模板，并通过平移、轻微扰动生成合成样本。
- 为每个样本加入高斯噪声，形成带噪输入和干净目标图像。
- 实现 `mean_filter` 均值滤波基线方法。
- 使用 Numpy 实现单隐层 `denoising_autoencoder`，包含编码层、解码层和反向传播训练。
- 统计重建 MSE、PSNR、边缘对比度和模板识别准确率。
- 生成重建效果网格图、训练损失曲线、像素误差分布图和指标对比图。
- 输出 `denoising_metrics_by_sample.csv` 和 `metrics.json`，方便复现实验和写提交说明。

## 运行方法

在本目录执行：

```bash
python main.py
```

也可以指定输出目录：

```bash
python main.py --output assets
```

运行后会生成：

- `reconstruction_grid.png`：干净图、带噪图、均值滤波结果和自编码器重建结果对比。
- `training_loss.png`：自编码器训练过程中的 MSE 损失曲线。
- `pixel_error_distribution.png`：两种去噪方法的像素误差分布。
- `metric_comparison.png`：MSE、PSNR、边缘对比度和模板识别准确率对比。
- `denoising_metrics_by_sample.csv`：每个测试样本的重建误差。
- `metrics.json`：核心指标和优化百分比。

## 项目意义

该模块贴近神经网络课程中的自编码器、表示学习和反向传播训练。自编码器通过压缩隐藏层学习数字图像的主要结构，能够把随机噪声映射回更接近数字模板的图像。这个项目可以继续扩展到卷积自编码器、变分自编码器、MNIST 数据集、异常检测和图像修复任务。

## 测试

```bash
python -m pytest tests
```

测试内容包括：

- 自编码器是否降低重建误差并提升 PSNR。
- 训练损失是否随迭代下降。
- 主程序是否能生成指标文件和全部可视化图片。
