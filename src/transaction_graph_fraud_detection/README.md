# 交易网络欺诈检测与图特征分类

本项目实现了一个面向金融风控的交易网络欺诈检测模块。普通账户风险识别往往只看单个账户的统计特征，例如交易次数、交易金额、夜间交易比例等；但欺诈账户通常还会呈现网络结构特征，例如与其他高风险账户频繁互转、形成局部团伙、二跳邻居异常密集等。本项目在账户统计特征基线基础上，引入交易图邻居聚合特征和局部结构特征，用逻辑回归分类器比较两种特征体系的欺诈识别效果。

## 功能内容

- 构建 180 个账户和多条转账交易组成的合成交易网络。
- 为账户设置用户分群、账户年龄和欺诈标签。
- 模拟正常账户与欺诈账户在交易金额、交易时间、跨分群交易和团伙互转上的差异。
- 提取基础统计特征，包括出入度、交易金额、夜间交易比例、跨分群比例和账户年龄。
- 提取图增强特征，包括邻居均值聚合、交易图度数、二跳邻居数量和局部聚集系数近似值。
- 实现带类别权重的逻辑回归分类器。
- 统计准确率、精确率、召回率、F1、AUC-like 排序分数和 Top-K 召回率。
- 生成交易图投影、欺诈概率分布、邻接矩阵热力图和指标对比图。
- 输出 `fraud_scores.csv` 和 `metrics.json`，方便复现实验和写提交说明。

## 运行方法

在本目录执行：

```bash
python main.py
```

也可以指定输出目录：

```bash
python main.py --output assets
```

运行后会生成：

- `transaction_graph_projection.png`：交易网络投影图，标出正常账户和欺诈账户。
- `fraud_score_distribution.png`：图增强模型输出的欺诈概率分布。
- `metric_comparison.png`：基础统计特征与图增强特征的检测指标对比。
- `adjacency_heatmap.png`：账户交易邻接矩阵热力图。
- `fraud_scores.csv`：测试账户真实标签和两种模型的风险分数。
- `metrics.json`：核心指标和优化增量。

## 项目意义

该模块贴近机器学习课程中的图特征学习和分类任务。虽然没有依赖图神经网络框架，但保留了图学习中的核心思想：节点不仅由自身属性决定，也受到邻居结构和局部网络模式影响。图增强特征可以作为 GCN、GraphSAGE 或图注意力网络的前置解释版本，适合课程项目展示和后续扩展。

## 测试

```bash
python -m pytest tests
```

测试内容包括：

- 图增强特征是否提升 F1、排序分数或 Top-K 召回。
- 图特征矩阵维度是否与账户数量一致。
- 主程序是否能生成指标文件和全部可视化图片。
