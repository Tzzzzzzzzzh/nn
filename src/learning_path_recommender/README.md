# 学生练习推荐与知识掌握度矩阵分解

本项目实现了一个面向在线学习平台的学生练习推荐模块。学生在学习线性代数、概率论、优化方法和神经网络等知识点时，答题记录通常是不完整的：每个学生只做过一部分题目，系统需要根据已有答题结果推断学生对不同知识点的掌握程度，并推荐难度合适的下一批练习。本项目在知识点平均正确率基线基础上，实现矩阵分解模型，用低维学生向量和题目向量预测未做题目的正确概率。

## 功能内容

- 构建 72 名学生、50 道练习题、5 个知识点的合成答题数据。
- 为每道题设置知识点标签和难度参数。
- 模拟学生真实知识掌握度，并生成不完整答题记录矩阵。
- 实现 `concept_average` 知识点平均正确率基线方法。
- 实现 `matrix_factorization` 矩阵分解推荐方法，学习学生隐向量、题目隐向量和偏置项。
- 预测未观测题目的正确概率，并为学生推荐成功率接近目标挑战区间的练习。
- 统计 RMSE、准确率、AUC-like 排序分数、知识掌握度误差和推荐难度匹配度。
- 生成答题矩阵图、预测掌握度热力图、推荐题目得分图和指标对比图。
- 输出 `recommendations.csv` 和 `metrics.json`，方便复现实验和写提交说明。

## 运行方法

在本目录执行：

```bash
python main.py
```

也可以指定输出目录：

```bash
python main.py --output assets
```

运行后会生成：

- `response_matrix.png`：学生-练习答题矩阵，展示缺失值、答错和答对记录。
- `predicted_mastery_heatmap.png`：矩阵分解模型预测的学生知识点掌握度。
- `recommendation_scores.png`：前 8 名学生的推荐题目成功概率。
- `metric_comparison.png`：基线方法与矩阵分解方法的指标对比。
- `recommendations.csv`：推荐题目、知识点、难度和预测成功率。
- `metrics.json`：核心指标和优化百分比。

## 项目意义

该模块贴近神经网络课程中的表示学习和推荐系统思想。矩阵分解可以看作一个浅层神经推荐模型：学生和题目被映射到低维隐空间，二者内积表示匹配程度。相比只看知识点平均正确率，矩阵分解能够捕捉学生之间和题目之间的潜在相似性，从而更准确地预测未做题目的表现。

该项目可以继续扩展为神经协同过滤、知识追踪、图神经网络推荐、遗忘曲线建模和自适应学习路径规划。

## 测试

```bash
python -m pytest tests
```

测试内容包括：

- 矩阵分解是否降低预测 RMSE 并提升分类/排序指标。
- 推荐题目是否都是学生未做过的练习。
- 主程序是否能生成指标文件和全部可视化图片。
